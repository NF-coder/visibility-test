# Visibility API & Focus/Blur Test

A simple tool for visually testing and comparing two important browser APIs: **Page Visibility API** and **Focus/Blur events**.

- **Visibility API** - detects whether the user can see the current tab
- **Focus/Blur** - detects whether the page or element has input focus
